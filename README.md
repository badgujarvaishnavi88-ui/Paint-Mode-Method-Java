# Paint Mode Demo - Java Applet

## 📌 Description
This project demonstrates the use of **Paint Mode in Java Applet** using the **AWT Graphics class**.

The program creates a simple graphical interface where it draws a rectangle, oval, and text with different colors using the `Graphics` object.

## 🚀 Features
- Uses Java Applet
- Demonstrates `paint()` method
- Uses AWT Graphics for drawing
- Draws rectangle and oval shapes
- Displays text using Graphics
- Applies different colors using Color class

## 🛠️ Technologies Used
- Java
- Java Applet
- AWT (Abstract Window Toolkit)

## 📂 Project Structure
PaintModeDemo/
│
└── PaintModeDemo.java

## 💻 Code Explanation

- `paint(Graphics g)` method is used to perform drawing operations.
- `setPaintMode()` sets the default painting mode.
- `setColor()` changes the drawing color.
- `drawRect()` creates a rectangle.
- `drawOval()` creates an oval shape.
- `drawString()` displays text on the applet window.
